توضیحات کامل ماد

ماد هلیکوپتر AH-64D Apache یک ماد فوق‌حرفه‌ای و کاملاً واقعی برای بازی GTA V است که هلیکوپتر نظامی پیشرفته AH-64D Longbow Apache را با تمامی جزئیات به بازی اضافه می‌کند.

این ماد با دقت بسیار بالا طراحی شده و شامل تمامی ویژگی‌های هلیکوپتر واقعی Apache می‌باشد. از سیستم تسلیحات کامل گرفته تا کابین خلبان با نمایشگرهای واقعی، صداهای اصلی موتور و شلیک، و سیستم آسیب‌پذیری دقیق.

پیش‌نیازهای سیستم

حداقل نیازمندی‌های سیستم برای اجرای ماد

نیازمندی سیستم

  • سیستم‌عامل: Windows 10 64 Bit, Windows 8.1 64 Bit, Windows 8 64 Bit, Windows 7 64 Bit Service Pack 1, Windows Vista 64 Bit Service Pack 2
  • CPU (پردازنده): Intel Core 2 Quad CPU Q6600 @ 2.40 GHz / AMD Phenom 9850 Quad-Core Processor @ 2.5 GHz
  • مموری: ۴GB RAM
  • گرافیک: NVIDIA 9800 GT 1GB / AMD HD 4870 1GB
  • دایرکت‌ایکس: DirectX 10

راهنمای نصب:

  1. وارد پوشه بازی شوید ، فولدر Update بازی را در فولدر Mods کپی کنید (اگر فولدر Mods ندارید دستی بسازید)
  2. ماشین هایی که با روش ADD-On نصب میشوند معمولا دارای یک فولدر و یک فایل تکست هستند
  3. فولدر (ah64d) را در آدرس روبرو کپی کنید : XXX:\Grand Theft Auto V\mods\update\x64\dlcpacks
  4. وارد برنامه Open IV شوید ، به آدرس روبرو بروید : mods / update / update.rpf / common / data
  5. دکمه F6 رو بزنید تا Edit Mod فعال شود
  6. فایل dlclist.xml را پیدا و روی آن کلیک راست کرده و Edit را بزنید
  7. این خط رو مثل بقیه خط ها اضافه کنید:
    <Item>dlcpacks:\ah64d\</Item>
  8. روی save بزنید و بازی را اجرا کنید و از آن لذت ببرید
  9. نکته: میتوانید از آموزش ویدیویی استفاده کنید!
